# The ANSI variant of the Keychron Q1

## Function Layer:
    space    : Play / Pause Track
    b        : Previous Track
    n        : Next Track
    m        : Mute / Unmute
    <        : Volume Down
    >        : Volume Up
    up       : layer ++  (+2)    ???
    down     : layer --  (-2)    ???
    
## Tap dance
    super
        x2   : File manager (super + e)
        x2_  : terminal (control + alt + t)
    fn  +  esc
        x3_  : Sleep mode
    fn
        x1   : enables OSL on layer two
        x1_  : layer two 
    home
        x2(_): Page Down
    end
        x2(_): Page Up
    caps lock
        x1_  : Changes language
